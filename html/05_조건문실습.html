홀 / 짝 확인하기

if (만약에) - else if(아니라면 이렇게, 저렇게) else(모두 아니면)

        
            if(조건식1){
                조건식1이 true인 경우에만 실행
            }
            else if(조건식 2){
                위의 if가 false이면서 조건식2가 true인 경우
            }
            else{
                위의 조건문이 모두 false일 때
            }
        

        *else if 구문은 원하는 만큼 작성 가능

        *앞선 if/else if문 조건이 false인 경우
            이후 else if에서 해당 범위는 제외
